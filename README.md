## Technology Stack

Language and Framework:
- C#
- ASP.NET Core 6

Database:
- MS SQL Server
- In-memory database (for tests)

Architecture Pattern:
- Clean Architecture

Design Pattern:
- CQRS Design Pattern
- Mediator Design Pattern
- Repository Design Pattern

Feature:
- Swagger UI
- Entity Framework
- MediatR
- AutoMapper
- Fluent Validation
- Quartz
- Serilog

Authentication:
- JWT Authentication

Testing:
- xUnit
- Moq
- Fluent Assertions
- Auto Fixture
